# Vani Tuition Center System – Prototypes Repository

## Purpose
This repository contains all UI/UX design prototypes, wireframes, and visual design assets related to the Vani Tuition Centre Admin Portal and Website System.

The repository is used during the design phase of the System Design Project to manage prototype versions, design drafts, and approved UI concepts.

---

## Repository Structure

```plaintext
prototypes/
│
├── Wireframes/
├── Low-Fidelity-Prototypes/
└── Figma-Exports/
```

## Branch Strategy

| Branch | Purpose |
|--------|---------|
| `main` | Approved prototype designs only |
| `draft/<prototype-name>` | Prototype development branches |
| `fix/<issue-number>` | Design corrections and review fixes |


---

## Contribution Guidelines

All contributors must follow:

- Branch naming conventions
- Commit message conventions
- Pull request review process

Please read `CONTRIBUTING.md` before making changes.

---

## Team Workflow

1. Pull the latest repository changes
2. Create a prototype draft branch
3. Add or modify design files
4. Commit changes using proper commit conventions
5. Push the branch to GitHub
6. Create a Pull Request
7. Receive team approval
8. Merge into `main`
